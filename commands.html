Introduction

Welcome to the page where you can find all of the commands, even upcoming commands for the bot. Special thanks to Pure.css for providing this page. This page will be updated frequently when a new version is LIVE on the bot or new commands for the upcoming update/version. The bot has a total of 5 main modules which are Moderation, Misc/Fun, Music, Games and Custom. There is a dashboard for the custom modules but you can still customize certain commands without heading to the dashboard by typing the command! You can find those under Custom Modules.
